# k2i

Kernel Parameters Interface using HTTP

## Quick Install

```
cargo install k2i
```

## k2i Server Run

```
k2i 
```

## Support API

- root response

```
$ curl 127.0.0.1:8000/k2i/api/v1/
root response!
```

- test response

```
$ curl 127.0.0.1:8000/k2i/api/v1/hello
hello!
```
- current configuration response

```
$ curl -s 127.0.0.1:8000/k2i/api/v1/config | jq .
{
  "hostname": "127.0.0.1",
  "port": 8000,
  "threads": 1
}
```
